Paste structured markdown below and click Generate. The builder produces a branded PPTX using the HRI template with correct layouts, colors, and master elements.

Frontmatter (Optional)

---
title: Deck Title
subtitle: Optional Subtitle
date: April 2026
---

If present, generates a TITLE slide automatically as the first slide.

Content Rules

  • Each slide starts with # KEYWORD: Title Text
  • Bullets start with -
  • KPI values use - VALUE | LABEL
  • TWO-COL uses left: and right: sections
  • SECTION accepts color: tide|golden|restore|risingsun
  • QUOTE uses quote: and attribution:
  • Unknown keywords default to CONTENT
  • Empty slides (no content) are skipped
